# Requirements
 
## install
 
### Node.js 라이브러리 설치
 ```js
 $ npm install
 ```
 
### pngjs 설치
 ```js
 $ npm i pngjs
 ```

 ### graceful-fs 설치
 ```js
 $ npm i graceful-fs
 ```

 ### svgicons2svgfont 설치
 ```js
 $ npm i svgicons2svgfont
 ```

## dir
[fonts](/fonts) : customfomt.ttf 저장되는 곳  
[save_dir](/save_dir) : inferrence 결과가 유니코드.png로 저장되어 있는 곳  
[svgs](/svgs) : 문자 각각이 유니코드.svg로 저장되는 곳  
[svg_font](/svg_font) : 문자 각각에 대한 svg파일을 하나로  


## run
### Make Font
```js
$ node generate_ttf.js
```
